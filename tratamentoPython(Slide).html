Jogo de Adivinhação - CORRIGIDO

import random

numero_secreto = random.randint(1, 10)
print("Adivinhe o número entre 1 e 10!")

try:
    palpite = int(input("Digite seu palpite: "))
except ValueError:
    print("❌ Erro! Digite apenas NÚMEROS inteiros!")
else:
    if palpite == numero_secreto:
        print("Parabéns! Você acertou!")
    else:
        print(f"Errou! O número era {numero_secreto}")

💡 Qual a diferença?

O programa não quebra mais quando o usuário erra a digitação!

🛠️ Corrigindo a Calculadora

Calculadora - CORRIGIDA

print("Calculadora: 10 dividido por X")

try:
    x = int(input("Digite o valor de X: "))
    resultado = 10 / x
except ValueError:
    print("❌ Digite apenas números!")
except ZeroDivisionError:
    print("❌ Não é possível dividir por zero!")
else:
    print(f"10 / {x} = {resultado}")

⚠️ Atenção!

Usamos DOIS except para erros diferentes!

🛠️ Corrigindo a Lista de Compras

Lista de Compras - CORRIGIDA

compras = ["Arroz", "Feijão", "Macarrão"]

try:
    indice = int(input("Digite o índice (0 a 2): "))
    print(f"Produto: {compras[indice]}")
except ValueError:
    print("❌ Digite um número inteiro!")
except IndexError:
    print("❌ Índice inválido! Digite 0, 1 ou 2.")

⚙️ ATIVIDADE 2: Tratamento de Exceções

Estrutura básica:

try:
    # Código que PODE dar erro
    numero = int(input("Digite um número: "))
except ValueError:
    # Executa se der erro
    print("Isso não é um número válido!")
else:
    # Executa se NÃO der erro
    print(f"O dobro é {numero * 2}")

Lembrete: O `else` é opcional, mas muito útil!

Resumo da Aula
