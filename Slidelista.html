📊 Mas o que são listas?

Em programação, listas são uma coleção de itens em uma ordem específica.


Você pode criar uma lista com:

  • 📝 Letras do alfabeto
  • 🔢 Números de 1 a 100
  • 🛒 Produtos de supermercado
  • 🎉 Convidados de uma festa

Em Python, você pode colocar qualquer dado em uma lista, inclusive dados que não estão relacionados.


Para criar uma lista, utilize colchetes ([ ]) e separe com vírgulas os itens que serão armazenados.

🎉 Criando uma Lista de Convidados

convidados = ["José", "Lucas", "Ana", "Enzo", "Valentina"]

print(convidados)
Saída: ['José', 'Lucas', 'Ana', 'Enzo', 'Valentina']

Observe que, quando uma lista é impressa, é exibida uma representação da lista, incluindo os colchetes.

Porém, há outras formas para exibir tais itens...

⚙️ Operações com Listas

Operação Código Resultado
Criar lista lista = [1, 2, 3] [1, 2, 3]
Acessar item lista[0] 1
Adicionar no final lista.append(4) [1, 2, 3, 4]
Adicionar em posição lista.insert(1, 99) [1, 99, 2, 3, 4]

➕ Método append()

O método append() adiciona um elemento no FINAL da lista.

lista = ["João", "Maria", "Felipe"]
lista.append("Danilo")

print(lista)
# Saída: ['João', 'Maria', 'Felipe', 'Danilo']
Antes: [João, Maria, Felipe]
Depois: [João, Maria, Felipe, Danilo]

Regra: append() sempre adiciona na ÚLTIMA posição!

📌 Método insert()

O método insert() adiciona um elemento em qualquer posição da lista.

lista = ["João", "Maria", "Felipe"]
lista.insert(1, "Danilo")  # posição 1, valor "Danilo"

print(lista)
# Saída: ['João', 'Danilo', 'Maria', 'Felipe']
Antes: [João, Maria, Felipe]
Depois: [João, Danilo, Maria, Felipe]

Sintaxe: insert(posição, valor)

🗑️ Método pop()

O método pop() remove um elemento pelo índice e retorna o valor removido.

lista = ["João", "Maria", "Felipe", "Danilo"]
removido = lista.pop(1)  # remove o índice 1 (Maria)

print(f"Removido: {removido}")
print(f"Lista atual: {lista}")
# Saída:
# Removido: Maria
# Lista atual: ['João', 'Felipe', 'Danilo']
Antes: [João, Maria, Felipe, Danilo]
Removeu: Maria (índice 1)
Depois: [João, Felipe, Danilo]

Sem índice: pop() remove o ÚLTIMO elemento.

📌 Organizando Listas

Problema: O usuário pode inserir números em qualquer ordem!

# Exemplo: usuário digita 1, 5, 2
numeros = [1, 5, 2]  # Está desordenado!

Solução: Python tem métodos para ordenar listas!

📊 Método sort()

O método sort() altera a posição dos itens para colocá-los em ordem:

  • Textos: ordem alfabética (A → Z)
  • Números: ordem crescente
convidados = ["Carlos", "Ana", "Bruno", "Daniel"]

# Ordem crescente (A → Z)
convidados.sort()
print(convidados)  # ['Ana', 'Bruno', 'Carlos', 'Daniel']

📊 Função sorted()

A função sorted() exibe os itens em ordem sem alterar a lista original.

numeros = [3, 1, 4, 1, 5, 9]

# Exibir ordenado sem modificar a lista
print(sorted(numeros))     # [1, 1, 3, 4, 5, 9]

# Lista original continua igual!
print(numeros)  # [3, 1, 4, 1, 5, 9]

🔄 Método reverse()

O método reverse() simplesmente inverte a ordem da lista (último vira primeiro).

🚗 Estacionamento: Carro1 → Carro2 → Carro3 → Carro4
🚗 reverse(): Carro4 → Carro3 → Carro2 → Carro1 
carros = ["Fusca", "Civic", "Corolla", "Gol"]

carros.reverse()
print(carros)  # ['Gol', 'Corolla', 'Civic', 'Fusca']

🔢 O que são Matrizes?

As matrizes estão presentes em jogos de tabuleiro (como Campo Minado e Xadrez), em planilhas eletrônicas, entre outros.


Por exemplo, a planilha abaixo poderia ser implementada com duas listas com seis itens cada.

📊 Exemplo: Planilha

A B
Nome Time
José Palmeiras
Maria Flamengo
Tiago Sport
Pablo Treze
Luana Cruzeiro

Esta planilha pode ser implementada com uma matriz (lista de listas) em Python!

💻 Código da Matriz

Veja o código necessário para armazenar os dados da planilha anterior:

timesXPessoas = [
    ["Palmeiras", "Flamengo", "Sport", "Treze", "Cruzeiro"],
    ["José", "Maria", "Tiago", "Pablo", "Luana"]
]

print(timesXPessoas[1][2])  # Mostra: Tiago
print(timesXPessoas[0][2])  # Mostra: Sport

📍 Acessando Matrizes

Para acessar valores de uma matriz:

matriz[linha][coluna]

  • Primeiro: índice da lista armazenada (linha)
  • Depois: índice do item que você quer acessar (coluna)
# No exemplo:
# timesXPessoas[1][2] → linha 1, coluna 2 → "Tiago"
# timesXPessoas[0][2] → linha 0, coluna 2 → "Sport"
